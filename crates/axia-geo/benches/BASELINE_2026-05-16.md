# AXiA-geo Practicality Bench Baseline (2026-05-16)

**Anchor**: Tier 2-B bench suite expansion (LOCKED #44 — complete meaning per merge).
**Capture commit**: `edfa14c` (main after PR #57 Tier 1 cleanup).
**Run**: `cargo bench --bench practicality_bench` (release mode).
**Platform**: Windows, dev machine. Single-run measurement (no statistical robustness yet — Criterion is future work).

## Purpose

1. **Perf regression detection** baseline for future architectural changes (mesh.rs split, NURBS Boolean optimization, etc.)
2. **Path A vs Path B quantification** — kernel-native ADR-094 / ADR-104 가치 정량 anchor
3. **Memory footprint baseline** for 1000-face mesh
4. **Invariant verification cost** for ADR-007 enforcement budget
