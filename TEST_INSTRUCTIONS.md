## 🧪 테스트 체크리스트

### **페이지 로드 (필수)**
- [ ] 페이지가 검은 배경으로 로드됨
- [ ] 메뉴바가 상단에 표시됨 (File, Edit, View, Help)
- [ ] 3D 그리드가 보임
- [ ] 콘솔에 에러가 없음 (F12 > Console)

### **3D 뷰포트**
- [ ] 마우스 좌측 드래그: 카메라 회전
- [ ] 마우스 중간 드래그: 팬(이동)
- [ ] 마우스 휠: 줌 인/아웃
- [ ] 그리드가 정상적으로 표시됨

### **도구 (Tools)**
#### Line Tool (선 그리기)
```
단축키: L
1. 3D 뷰포트에서 클릭
2. 다시 클릭하여 끝점 설정
```
- [ ] 선이 올바르게 그려짐
- [ ] 스냅 지점이 표시됨

#### Rectangle Tool (사각형)
```
단축키: R
1. 첫 번째 점 클릭
2. 두 번째 점 클릭하여 사각형 생성
```
- [ ] 사각형이 올바르게 그려짐

#### Circle Tool (원)
```
단축키: C
1. 중심점 클릭
2. 반지름까지의 거리 지정
```
- [ ] 원이 올바르게 그려짐

### **3D 조작**
#### Push/Pull (면 밀어내기)
```
단축키: P
1. 면 클릭
2. 마우스 이동하여 높이 지정
3. 클릭하여 확정
```
- [ ] 3D 면이 생성됨
- [ ] 고스트 프리뷰가 표시됨

#### Move (이동)
```
단축키: M
1. 객체 선택
2. 시작점 클릭
3. 종료점 클릭
```
- [ ] 객체가 이동됨

#### Rotate (회전)
```
단축키: O
1. 객체 선택
2. 회전 중심 지정
3. 회전각 입력
```
- [ ] 객체가 회전됨

### **파일 I/O**
- [ ] File > Save: 파일 저장 가능 (.xia)
- [ ] File > Open: 파일 열기 가능
- [ ] File > Import: DXF 등 가져오기 가능

### **UI 패널**
- [ ] 우측 Inspector 패널 표시
- [ ] 좌측 Snap 옵션 패널
- [ ] 상단 도구 버튼 표시

### **버그 체크**
- [ ] 콘솔 에러 없음
- [ ] 마우스 클릭 반응 지연 없음
- [ ] 렌더링 끊김 없음
- [ ] 메모리 누수 없음

## 🔍 콘솔 에러 확인

**F12 키를 눌러 개발자 도구를 열고:**

1. **Console 탭** 확인
2. 빨간색 에러 메시지가 있으면 스크린샷 촬영
3. 메시지 내용을 보고해주세요
