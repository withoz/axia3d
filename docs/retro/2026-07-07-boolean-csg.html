핵심 아키텍처 결정 ADR-277 · 278

shared-vertex imprint

교차 세그먼트를 각 면에 각인 + add_vertex dedup → seam 구성상 봉합 (weld 불필요)

Pattern 12 재사용

boundary_kernel::arrange(기존 planar arrangement)로 가장 어려운 조각 — constrained subdivision + coplanar 최소 세분 — 해결

v2-first + v1 fallback

boolean_solid이 v2 먼저 시도, 실패 시 v1 → 무회귀 cutover. UI가 자동으로 회전·일반 CSG 획득

ADR-278 β · polygonalize-then-CSG

Path B 곡면 operand를 진입점에서 1회 polygonalize → v2·v1 둘 다 다면체를 봄. v2 내부에 두면 rollback 시 소실되어 v1이 원본 analytic 면을 받음 → 배치가 핵심.
